# Promise

> 一步步自己实现简单的 Promise



## 目录

| 文件名     | 描述                                 |
| ---------- | ------------------------------------ |
| basic      | 基本的可使用 then 方法回调的 Promise |
| chaining   | 可多次调用 then 方法添加多个回调     |
| rejectable | 可使用 catch 方法添加拒绝状态的回调  |
| exception  | 可使用 catch 方法处理错误            |
| method-all | 实现 Promise.all 方法                |



## 测试

运行 `test` 目录下的测试脚本：

```
node test/basic
```
